Linguagem de marcação

O HTML é uma lingagem de marcação utilizada para definir a estrutura do conteúdo. Ele consiste de uma séreie de elementos usado para delimitar ou agrupar diferentes partes de conteúdo. Esses elementos são denominados tags que servem desde a identificação dos dados do site para leitura pelos buscadores de sites e conteúdos até a definição de palavras em negrito ou itálico. A maioria das tags possuem abertura e fechamento identificada pelo uso de parênteses angulares.

Estrutura básica do HTML

A estrutura principal de uma página feita com HTML consiste da própria definição da linguagem de marcação (HTML), o head que inclui as informações de título e metadadospara otimização de conteúdo, o body que trata do conteúdo principal do site e o fechamento da tag html.

Marcações de texto

Os textos que compõem o cabeçalho do conteúdo são identifcados pelas tags de < h1 > até < h6 > a exemplo do que ocorrem em um índice de um livro. Cada uma dessas tags funciona como um nível de cabeçalho, como se vê no exemplo a seguir:

  • Título em H1

  • Título em H2

  • Título em H3

  • Título em H4

  • Título em H5
  • Título em H6

Principais tags de texto

Os parágrafos são identificados pela tag < p > intercaladas pelo seu conteúdo. No decorrer do texto podem ser utilizadas outras tags que indicam uma marcação, como < mark > que deixa o texto grifado assim.

Também podem ser feitas as marcações para negrito com o uso de < strong > ou itálico com o uso da tag < i >. Existem também as tags para definir letras subescritas, tais como números usados em elementos químicos como H2O, ou sobrescritas como no caso de marcas registradas que levam o termo tm.

Existem também outras tags usadas para definir o estilo de um determinado trecho, quando colocada após a tag style para definir um tipo de fonte a ser usada no texto. Outra tag comum é utilizada para corrigir ou demontrar algum erro, dessa forma, com o uso da tag < del >. As palavras abreviadas, como é o caso de HTML, podem exigir o uso da tag < abbr >. Um trecho do texto também receber a tag < u > para ficar com essa aparência de sublinhado.

Listas ordenadas e desordenadas

Por fim, temos as duas principais tags usadas para uso em listas. Quando a lista é ordenada, recebe a tag OL, mas quando é desornada, recebe a tag UL. Os itens específicos de cada lista devem ser colocados entre as tags il, como no exemplo a seguir:

    Lista ordenada
  1. Item 1
  2. Item 2
  3. Item 3

    Lista desordenada
  • Item 1
  • Item 2
  • Item 3

Todas as tags de HTML estão descritas nesse documento da W3Schools.
